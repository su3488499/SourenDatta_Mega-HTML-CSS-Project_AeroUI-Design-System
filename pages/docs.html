Design tokens

Tokens live in css/tokens.css and power consistent spacing, type, and neon colors.

:root{
  --neon-cyan:#20f7ff;
  --space-5:24px;
  --r-lg:22px;
}
Tip:

Use tokens everywhere — don’t hardcode random colors.

Layout system

Use .container + .grid utilities for responsive layouts.

<div class="container">
  <div class="grid grid-3">...</div>
</div>
Note:

On small screens, grids auto collapse to one column.

Components

Buttons, cards, badges, forms, and tables live in css/components.css.

Accessibility

Use semantic HTML first. Prefer native elements like <details> for accordions.

Why use details/summary?
It’s keyboard accessible by default and needs no JavaScript.
Focus styling?
Inputs and buttons use visible focus rings for keyboard users.
